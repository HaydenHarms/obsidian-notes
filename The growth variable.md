# The growth variable
When people think about [[AI taking jobs]], they often restrict themselves to thinking very narrowly by limiting the growth variable in the equation. The argument often goes: if AI can do work humans do now, then there will not be as many humans. This is natrually intuitive, if a system can be modelled as y, productive output, d, demand, and x the amount of human workers, then the equation looks something like this:
```math
y=/frac{d){x}
```
